How to transfer settings → lib/ocr/tesseract.ts


        

          1
          
Click Copy config JSON above after finding a good config.

        


        

          2
          

            In lib/ocr/tesseract.ts, update extract():
            
// add OEM as second arg (0=Legacy, 1=LSTM, 2=Both)
const worker = await createWorker("eng", 1)

// set PSM + whitelist before recognize()
await worker.setParameters({
  tessedit_pageseg_mode: '11',   // ← from psm
  tessedit_char_whitelist: '',   // ← from whitelist
})

const { data } = await worker.recognize(buffer)

          

        


        

          3
          

            For scale / binarize / denoise, use sharp server-side before passing to Tesseract:
            
import sharp from 'sharp'
const processed = await sharp(buffer)
  .grayscale()
  .resize({ width: origW * 2, kernel: 'cubic' })   // scale 2×
  .threshold(128)                                    // binarize
  .toBuffer()

            PSM & OEM transfer directly; preprocessing needs sharp (already common in Next.js).
